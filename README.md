# react-iTunes-search

simple web app for iTunes search with React

## Preview

![preview gif](https://dl.dropboxusercontent.com/u/3896368/react-iTunes-search.gif)

## Installation

1. `git clone git@github.com:LeoAJ/react-iTunes-search.git`
2. `cd react-iTunes-search`
3. `npm install`
4. `gulp`

## Built with

* react
* babel
* ES6
* gulp
* webpack
* eslint
* semantic UI
* iTunes search API
