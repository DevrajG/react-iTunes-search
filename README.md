# react-iTunes-search

simple web app for iTunes search with React

## Installation

```
git clone git@github.com:LeoAJ/react-iTunes-search.git
cd react-iTunes-search
gulp
```

## Built with

* react
* babel
* ES6
* gulp
* webpack
* eslint
* semantic UI
* iTunes search API
